##### 软件使用
远程控制相关指令：
> *  #帮助
> *  #截图
> *  #关机
> *  #取消关机
> *  #打开@文件/程序                `如：#打开@d:\abc.txt   或者   #打开@d:\qq.lnk`
> *  #关闭@进程名                   `如：关闭@notepad      或者    #关闭@qq`
> *  #网页@www.baidu.com           `默认用IE浏览器打开，如有需要可以修改`
> *  #关闭浏览器
> *  #控制@cmd命令                  `如：#控制@explorer c:\windows`   
> *  #最小化窗口
> *  #切换窗口
